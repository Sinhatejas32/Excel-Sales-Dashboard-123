# Excel-Sales-Dashboard-123
This project focuses on analyzing retail sales data using Microsoft Excel to derive meaningful business insights and support data-driven decision-making. The objective is to understand sales performance, customer behavior, and product trends through structured data analysis and an interactive dashboard.

The project covers the complete data analytics workflow including data cleaning, transformation, pivot table analysis, KPI tracking, and dashboard visualization.
